# CAT 2024 Syllabus — Complete Breakdown

## Exam Overview

| Parameter | Details |
|-----------|---------|
| Full Name | Common Admission Test |
| Conducted by | IIMs (rotating) |
| Frequency | Once a year (November) |
| Duration | 120 minutes |
| Total Questions | 66 |
| Sections | 3 (VARC, DILR, QA) |
| Sectional time limit | 40 minutes each |
| Mode | Computer-Based Test (CBT) |
| MCQ marking | +3 correct, −1 wrong |
| TITA marking | +3 correct, 0 wrong (no negative) |

---

## Section 1: Verbal Ability & Reading Comprehension (VARC)

**Questions: 24 | Time: 40 min | Weightage: ~36%**

### 1.1 Reading Comprehension (RC)
4 passages × 3–4 questions each (~16 questions)

**Passage Types:**
- Philosophy and Abstract Reasoning
- Economics and Business
- Science and Technology
- Social Sciences and History
- Literary and Cultural

**Question Types:**
- Main Idea / Central Theme
- Inference (What can be concluded from...)
- Author's Tone and Attitude
- Specific Detail (Fact-based retrieval)
- Vocabulary in Context
- Logical Structure of Passage

**Difficulty:** Passages are long (400–700 words), dense, and often counter-intuitive. Questions test close reading, not general knowledge.

### 1.2 Verbal Ability (VA)
~8 questions (mix of MCQ and TITA)

**Sub-topics:**
| Topic | Format | Tips |
|-------|--------|------|
| Para-Jumbles | 4–5 sentences → correct order | TITA in recent years; look for logical connectors |
| Para-Summary | 4 options → best summary of paragraph | Eliminate too-specific and too-broad options |
| Odd Sentence Out | 4 sentences, find the misfit | The odd one breaks the logical flow |
| Sentence Completion | Fill blank with best option | Context + grammar both matter |
| Critical Reasoning | Strengthen / Weaken / Assumption | Less common; appears in some years |

## Section 2: Data Interpretation & Logical Reasoning (DILR)

**Questions: 22 | Time: 40 min | Weightage: ~33%**

### 2.1 Data Interpretation (DI)
~10–12 questions (2–3 sets)

**Data Formats:**
| Format | What's Tested |
|--------|--------------|
| Tables | % change, ratio, ranking, aggregation |
| Bar Charts | Comparison, stacked bars, grouped bars |
| Pie Charts | Central angles, actual values from % |
| Line Graphs | Trend, CAGR, slope interpretation |
| Caselets | Paragraph-form data → extract & calculate |
| Mixed Charts | Two formats combined (e.g., table + line graph) |

**Skills Required:**
- Fast arithmetic + approximation
- Unit conversion (crore, lakh, thousand)
- Percentage and ratio calculations
- CAGR estimation

### 2.2 Logical Reasoning (LR)
~10–12 questions (2–3 sets)

**Sub-topics:**
| Topic | Frequency | Core Skill |
|-------|-----------|-----------|
| Linear Arrangement | Very High | Constraint placement, adjacency |
| Circular Arrangement | High | Relative positions, symmetry |
| Grouping & Selection | High | If-then chains, elimination |
| Scheduling / Calendar | Medium | Day/slot matrix |
| Set Theory / Venn | Medium | Inclusion-exclusion |
| Blood Relations | Low–Medium | Family tree drawing |
| Direction & Distance | Low | Cardinal directions, displacement |
| Puzzles (Custom) | Medium | Novel multi-constraint problems |

**Note:** CAT DILR sets are increasingly novel — "standard" sets are rare. Flexibility and approach matter more than memorising formats.

## Section 3: Quantitative Aptitude (QA)

**Questions: 22 | Time: 40 min | Weightage: ~33%**

### 3.1 Arithmetic
~8–10 questions | Highest weightage sub-area

| Topic | Key Concepts |
|-------|-------------|
| Ratio & Proportion | Direct/inverse proportion, mixture & alligation |
| Percentages | % change, successive %, reverse % |
| Profit & Loss | SP/CP/MP, discount, % profit on CP vs SP |
| Simple & Compound Interest | SI/CI formulas, effective rate |
| Time, Work & Pipes | Work rate, efficiency, combined work |
| Time, Speed & Distance | Relative speed, trains/boats, average speed |
| Mixtures & Alligation | Rule of alligation, three-component mixing |

### 3.2 Number Systems
~3–4 questions

| Topic | Key Concepts |
|-------|-------------|
| Divisibility | Rules for 2,3,4,5,6,7,8,9,11 |
| HCF & LCM | Relationship: HCF × LCM = product (for 2 numbers) |
| Remainders | Cyclicity, Fermat's theorem |
| Unit Digit | Cyclicity of last digits |
| Factors | Number of factors, sum of factors |
| Prime Numbers | Sieve, properties |

### 3.3 Algebra
~4–5 questions

| Topic | Key Concepts |
|-------|-------------|
| Linear Equations | System of equations, word problems |
| Quadratic Equations | Vieta's formulas, nature of roots |
| Inequalities | Wavy curve method, modulus |
| Functions | Domain, range, special functions |
| Logarithms | Change of base, log rules |
| Progressions (AP/GP) | Sum formulas, nth term |
| Sequences | Recursive sequences, pattern recognition |

### 3.4 Geometry & Mensuration
~4–5 questions

| Topic | Key Concepts |
|-------|-------------|
| Lines & Angles | Parallel lines, transversal |
| Triangles | Congruence, similarity, area, Pythagoras |
| Circles | Chord-angle theorems, tangent-radius |
| Quadrilaterals | Properties of square, rectangle, rhombus, trapezoid |
| Coordinate Geometry | Distance, midpoint, slope, equation of line |
| 3D Mensuration | Cube, cuboid, cylinder, cone, sphere — SA and volume |

### 3.5 Modern Mathematics
~3–4 questions

| Topic | Key Concepts |
|-------|-------------|
| Permutation & Combination | nPr, nCr, circular, identical objects |
| Probability | Basic probability, conditional, Bayes' |
| Set Theory | Venn diagrams, inclusion-exclusion |

## Important Notes for CAT 2024

1. **TITA questions** (~30–35% of paper) have no negative marking — never leave them blank
2. **Sectional cutoffs** matter — scoring high in one section doesn't compensate for a low sectional percentile
3. **IIM Ahmedabad** weight: VARC > DILR > QA (approx); varies by IIM
4. **Ideal attempt**: 48–54 questions with 85%+ accuracy > 60 attempts with 70% accuracy
5. **Mock CATs matter more than topic study** after fundamentals are solid (>3 months in)
