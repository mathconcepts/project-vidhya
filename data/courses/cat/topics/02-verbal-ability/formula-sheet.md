# Verbal Ability — CAT Framework Sheet

## Para-Jumble Decision Tree

```
Step 1: Read all sentences quickly (30 sec)
   ↓
Step 2: Find the OPENING sentence
   → No pronoun reference to prior context
   → Introduces main concept with "a/an" (first mention)
   → Often sets the scene, time, or topic
   ↓
Step 3: Find the CLOSING sentence
   → Draws a conclusion or implication
   → Uses "thus", "therefore", "hence", "ultimately"
   → Or: poses a rhetorical question or future outlook
   ↓
Step 4: Build PAIRS using connectors
   → "however" / "but" / "yet" → contrast, follows positive claim
   → "therefore" / "thus" / "hence" → conclusion, follows reasoning
   → "moreover" / "furthermore" / "also" → addition, follows similar idea
   → "for example" / "for instance" → follows a general claim
   → "this" / "these" / "it" / "they" → follows the referenced noun
   ↓
Step 5: Assemble the chain and verify coherence
```

## Connector Reference Card

### Contrast Connectors (cannot open a paragraph)
- However, But, Yet, Although, Even though, Despite, Nevertheless, Nonetheless, On the contrary, In contrast, Whereas

### Addition Connectors
- Moreover, Furthermore, Additionally, Also, Besides, In addition, What is more

### Conclusion/Result Connectors
- Therefore, Thus, Hence, Consequently, As a result, So, Accordingly

### Example Connectors (follow a general claim)
- For example, For instance, Such as, To illustrate, Specifically

### Sequence Connectors
- First, Second, Then, Next, Finally, Subsequently, Meanwhile

## Para-Summary Elimination Framework

| Option Type | How to Spot | Eliminate? |
|------------|-------------|-----------|
| Too specific | Focuses on one example/detail from the paragraph | ✅ Eliminate |
| Too broad | Claims more than the paragraph argues | ✅ Eliminate |
| Contradictory | Reverses the paragraph's stance | ✅ Eliminate |
| External info | Introduces information not in the paragraph | ✅ Eliminate |
| Correct | Captures the main argument, neutral scope, same tone | ✅ Keep |

### Summary Quality Checklist
- [ ] Does it answer: "What is the paragraph mainly saying?"
- [ ] Is every word justified by paragraph content?
- [ ] Does it avoid extreme words (always, never, all, none) unless passage uses them?
- [ ] Does it match the passage's tone (analytical, critical, descriptive, argumentative)?

## Odd Sentence Out — Identification Framework

```
1. Read all sentences
2. Identify the common THEME in 3+ sentences
3. Check the "outlier" sentence:
   - Does it introduce a new subject not mentioned elsewhere?
   - Does it break the logical/temporal flow?
   - Does it contradict what others establish?
   - Does it use pronouns that don't connect to the others?
4. The sentence that breaks the theme/flow = ODD SENTENCE
```

### Common Odd-Sentence Patterns
- Sentences about Topic A's history, but odd sentence is about Topic A's future
- Sentences from one perspective, but odd sentence uses a different voice
- All sentences discuss a problem, but odd sentence discusses an unrelated solution

## Grammar Quick Reference

### Subject-Verb Agreement Edge Cases
| Construction | Rule | Example |
|-------------|------|---------|
| Neither...nor | Verb agrees with nearest noun | Neither the students nor the teacher **is** ready |
| Either...or | Same as above | Either the plans or the budget **needs** revision |
| Each/Every | Always singular | Each of the boys **is** here |
| Collective nouns | Usually singular | The committee **has** decided |
| Data/Criteria | Technically plural (but singular increasingly accepted) | The data **are/is** conclusive |

### Articles
- "a" before consonant sound: a university, a one-time event (u and o sound like consonants here)
- "an" before vowel sound: an hour, an MBA, an honest person
- "the" for unique items: the sun, the Prime Minister (specific, known)
- No article for plural generalisations: Dogs are loyal (not "The dogs")

### Parallelism Rule
All items in a list must use the same grammatical form:
- ❌ She likes running, to swim, and cycles
- ✅ She likes running, swimming, and cycling

## CAT-Specific VA Tips

1. **PJ opening clues**: Look for the sentence that defines or introduces the main concept without referring to anything prior
2. **PJ closing clues**: Sentences with "thus", "ultimately", "in the end" — or ones that give a final implication
3. **Para-summary**: If two options say similar things, the one that is slightly more specific to the passage wins
4. **Odd sentence**: When stuck, try removing each sentence mentally — the paragraph that reads most coherently after removal = that sentence is the odd one
5. **Reading speed**: Target 200-250 words per minute for active comprehension (not speed-reading) to handle VA comfortably within time
