## Communication Norms

### Squad Chat Rules
- Use task comments for work-related discussion (keeps context with the task)
- Squad chat for announcements only
- @mention specific agents, not broadcast

### Escalation Path
```
Sub-agent → Parent Agent → Jarvis → Giri (CEO)
```

### Handoff Protocol
1. Comment on task with context
2. @mention the receiving agent
3. Update task assignee
4. Receiving agent acknowledges
