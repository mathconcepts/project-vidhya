# Reading Comprehension — CAT Framework Sheet

## Question-Type Answer Framework

### Main Idea Questions
**Process:**
1. After reading, state the central claim in your own words
2. Eliminate: too narrow (details, examples), too broad (exceeds what passage argues), contradictory
3. Check: Does the remaining option work for ALL paragraphs in the passage?

**Correct option characteristics:**
- Captures the author's central argument/position
- Not just the topic ("This passage is about climate change" is not a main idea)
- Reflects the passage's scope — not broader, not narrower

---

### Specific Detail / Factual Questions
**Process:**
1. Find the keyword from the question in the passage
2. Read the 2-3 sentences around that keyword
3. Select the option that accurately paraphrases those sentences

**Traps to avoid:**
- Option uses exact passage words but changes a qualifier (e.g., "some" → "all", "often" → "always")
- Option cites a detail from a different part of the passage than the keyword appears

---

### Inference Questions
**The Inference Test:**
- Is the answer **directly supported** by passage content? (not: based on general knowledge)
- Is the answer **logically necessary** given what the passage says? (not just possible, but follows from evidence)
- Does the answer **avoid extremes** not supported by passage language?

**Correct inference = Passage is enough to prove it, and no more is needed**

---

### Tone & Attitude Questions
**Tone Vocabulary Reference:**

| Positive Tones | Negative Tones | Neutral Tones |
|----------------|----------------|---------------|
| Appreciative | Critical | Analytical |
| Admiring | Sceptical | Dispassionate |
| Optimistic | Cynical | Descriptive |
| Celebratory | Dismissive | Cautious |
| Sympathetic | Ironic | Objective |
| Nostalgic | Condescending | Ambivalent |

**Finding tone:**
- Look for evaluative adjectives and adverbs (clearly, unfortunately, remarkably, merely)
- Look for hedging language: "seems", "appears", "ostensibly", "supposedly" → author is sceptical
- Look for enthusiastic language: "remarkably", "crucially", "above all" → author endorses

---

### Vocabulary-in-Context Questions
**Process:**
1. Read the entire sentence containing the word
2. Without looking at options, determine what the word means in THIS context
3. Select the option closest to your contextual meaning (may differ from dictionary meaning)

**Example:** "The artist's technique was, by contemporary standards, primitive."
- Dictionary: 'primitive' = prehistoric
- Context: 'primitive' = simple/undeveloped (relative to contemporary standards)
- Correct answer: simple/rudimentary/unsophisticated

## Author Identification Cheat Sheet

### Signal Words That Reveal Author's Position
| Signal | What it indicates |
|--------|------------------|
| "Ironically", "paradoxically" | Author notes a contradiction they find meaningful |
| "Ostensibly", "supposedly", "purportedly" | Author is sceptical of this claim |
| "Crucially", "fundamentally", "above all" | Author endorses this as most important |
| "Merely", "simply", "only" | Author is minimising something |
| "Unfortunately", "regrettably" | Author disapproves |
| "Remarkably", "fascinatingly" | Author finds this noteworthy/impressive |
| "While X argues..., the reality is..." | Author is refuting X |

## Passage Structure Templates

### Argumentative Passage
```
Para 1: Main claim / thesis
Para 2: Evidence / examples supporting claim
Para 3: Counterargument (opposing view)
Para 4: Rebuttal + reinforced conclusion
```
**Main idea = thesis (Para 1), reinforced in Para 4**

### Problem-Solution Passage
```
Para 1: Problem described
Para 2: Why existing solutions fail
Para 3: Proposed solution
Para 4: Limitations / implications
```
**Main idea = the proposed solution and its context, NOT the problem alone**

### Historical/Descriptive Passage
```
Para 1: Historical context
Para 2-3: Development of the subject
Para 4: Current state / significance
```
**Main idea = the significance claim in the final paragraph, informed by the historical arc**

## Elimination Rules for RC Options

| Rule | Description |
|------|-------------|
| Extreme Language Rule | Eliminate options with "always", "never", "all", "completely" unless passage uses them |
| Outside Info Rule | Eliminate options that require knowledge not in the passage |
| Detail Rule | Eliminate options that accurately state a detail but miss the main/overall point |
| Contradiction Rule | Eliminate options that reverse or distort the passage's position |
| Comparison Rule | Between two close options, choose the one that is proportionally accurate — neither understating nor overstating |

## RC Time Budget (per passage)

| Activity | Time Allocation |
|----------|----------------|
| First read (active) | 2.5 – 3 min |
| Q1 (main idea) | 1.5 min |
| Q2 (specific detail) | 1 – 1.5 min |
| Q3 (inference) | 2 min |
| Q4 (tone/structure) | 1.5 min |
| **Total per passage** | **~9-10 min** |

## Active Reading Paragraph Cues

After each paragraph, ask:
1. **What does this paragraph do?** (introduces / supports / counters / concludes)
2. **What does the author think?** (approves / disapproves / is neutral)
3. **One-line takeaway**: "This paragraph argues that ___"

Build a mental map: Paragraph 1 (claim) → Paragraph 2 (evidence) → Paragraph 3 (counterargument) → Paragraph 4 (conclusion).
